#QueueLess

QueueLess is a full-stack web application built using Express.js and EJS.

## Features
* User Authentication(Staff & Visitor)
* Queue Management System
* Staff Dashboard
* Visitor Token System
* Firebase Integration

## Tech Stack
* Node.js
* Express.js
* EJS
* Firebase

## How to Run

1. Clone the repository
2. Run npm install
3. Add your firebase credentials
4. Run node server.js
